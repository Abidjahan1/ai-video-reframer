# Smart Video Reframer Studio

## Project Overview
A professional AI-powered desktop application for automatic video reframing, subject tracking, cinematic background generation, subtitle creation, and multi-platform export.

## Phase 1: Foundation
- Modular folder structure for scalability
- Main entry point (`main.py`)
- Core pipeline orchestrator (`VideoProcessor`)
- Centralized configuration and logging
- GUI controller stub

## How to Run
1. Ensure Python 3.11+ is installed.
2. Run the application:
   ```
   python app/main.py
   ```
